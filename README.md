# Fake Store App

A Flutter application with 3 screens: Splash, Login, and Home (Products list).

---

## Flutter Version Used

- Flutter: 3.35.7
- Dart: 3.9.2

---

## Steps to Run the Project

1. Clone the project
2. Run `flutter pub get`
3. Run `flutter run`
